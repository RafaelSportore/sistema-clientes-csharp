# 💻 Sistema de Cadastro de Clientes em C#

Projeto desenvolvido para praticar conceitos de programação orientada a objetos (POO) e manipulação de arquivos utilizando C#.

---

## 🚀 Funcionalidades

✔ Cadastrar clientes
✔ Listar clientes
✔ Buscar cliente por ID
✔ Atualizar dados do cliente
✔ Remover cliente
✔ Salvar dados em arquivo `.txt`
✔ Carregar dados automaticamente ao iniciar o sistema

---

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Listas (`List<T>`)
* Manipulação de arquivos (StreamWriter / File)
* Estruturas de repetição (`foreach`, `do while`)
* Validação de entrada (`TryParse`)

---

## 🛠 Tecnologias utilizadas

* C#
* .NET
* Console Application

---

## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos em lógica de programação e estrutura de sistemas CRUD, servindo como base para evolução para aplicações com interface gráfica e sistemas web.
